Nomor 8: Penjelasan AJAX dan jQuery

1. Pengertian AJAX

AJAX merupakan teknik dalam pengembangan website yang memungkinkan pertukaran data dengan server dilakukan tanpa perlu memuat ulang seluruh halaman. Dengan cara ini, tampilan web menjadi lebih cepat dan responsif karena hanya bagian tertentu saja yang diperbarui.

2. Pengertian jQuery

jQuery adalah pustaka (library) JavaScript yang dibuat untuk mempermudah penulisan kode. Dengan jQuery, berbagai aktivitas seperti memanipulasi elemen HTML, menangani event, membuat animasi, hingga penggunaan AJAX dapat dilakukan dengan lebih singkat dan mudah.

3. Cara Menggunakan jQuery

Agar bisa menggunakan jQuery, kita cukup menambahkan file library jQuery ke dalam halaman HTML. Cara yang paling praktis adalah menggunakan CDN (link online), sehingga tidak perlu menyimpan file secara manual di komputer.
