Kesimpulan

Berdasarkan hasil percobaan, tombol reset berfungsi untuk menghapus semua data yang telah diinputkan pada form sehingga kembali ke kondisi awal. Tombol kirim digunakan untuk mengirimkan data yang telah diisi ke server sesuai dengan atribut method yang digunakan. Atribut target berfungsi untuk menentukan di mana hasil pengiriman data akan ditampilkan, seperti _self (halaman yang sama) dan _blank (membuka di tab baru). Method GET menampilkan data pada URL sehingga mudah dilihat, sedangkan POST tidak menampilkan data pada URL sehingga lebih aman untuk data sensitif.
